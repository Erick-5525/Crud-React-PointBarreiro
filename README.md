📚 CRUD de Livros — React Puro
Projeto desenvolvido como atividade prática do curso de Front-End do POINT Barreiro.

📋 Sobre o Projeto
Aplicação web de gerenciamento de uma estante de livros, desenvolvida com React puro (sem bibliotecas externas de UI, sem Redux, sem Tailwind). O objetivo é demonstrar os conceitos fundamentais do React aplicados a um CRUD completo com persistência de dados.

✅ Funcionalidades

Criar — adicionar um novo livro com título, autor, ano, gênero e avaliação por estrelas
Ler — listar todos os livros com busca por título ou autor e filtro por gênero
Atualizar — editar qualquer informação de um livro já cadastrado
Deletar — remover um livro da lista
Persistência — os dados são salvos no localStorage e permanecem após fechar o navegador
Feedback visual — mensagem de confirmação aparece após cada ação (adicionar, editar, remover)
Ordenação — lista exibida em ordem alfabética pelo título
